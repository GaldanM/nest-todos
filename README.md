## Description

## Installation

```bash
$ yarn install
```

## Build the app

```bash
$ yarn run build
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

$ yarn run test:watch

# test coverage
$ yarn run test:cov

$ yarn run test:debug
```
